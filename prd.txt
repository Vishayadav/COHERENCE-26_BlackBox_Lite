--------------------------------------------------

1. PRODUCT OVERVIEW

OutreachFlow AI is a workflow‑driven automation platform that allows businesses to automate personalized outreach campaigns using AI‑generated messaging and configurable workflow logic.

The system allows users to:

• Upload or generate lead datasets
• Configure outreach workflows
• Generate personalized AI emails
• Automate follow-up sequences
• Monitor campaign performance through analytics dashboards
• Receive insights to improve conversion rates

--------------------------------------------------

2. PRODUCT OBJECTIVE

The platform aims to reduce manual outreach effort and provide a unified system for lead management, AI-driven outreach, and campaign performance monitoring.

Key Capabilities

• AI powered outreach generation
• Workflow based automation
• Lead management
• Conversion analytics
• Risk analysis and optimization insights

STAGE 2 — CAMPAIGN INTENT + PROMPT GENERATION
==================================================

Objective:
Collect the user's campaign goals, targeting intent, and personalization instructions.  
Use this information to generate personalized outreach email templates using Gemini.  
The AI must return emails strictly in CSV format containing subject and body fields.

--------------------------------------------------
Stage 2 Implementation
--------------------------------------------------

Step 1: Campaign Intent Form

Users must fill a structured form describing their outreach campaign.

Fields:

1. Campaign Name
Text Input
Example:
AI SaaS Founder Outreach

2. Target Audience
Text Input
Example:
Startup founders in SaaS companies

3. Product / Service Description
Text Area
Example:
AI powered customer support automation platform.

4. Value Proposition
Text Area
Example:
Reduce customer support workload by 60% using AI agents.

5. Goal of Outreach
Dropdown Options:
- Book a Demo
- Schedule a Call
- Partnership
- Sales Outreach
- Networking
- Custom

6. Personalization Variables
Multi-select checkboxes:

- First Name
- Company Name
- Job Title
- Industry
- Recent Achievement
- Location

These will be used as placeholders in the generated emails:

{{first_name}}
{{company}}
{{job_title}}
{{industry}}

--------------------------------------------------

Step 2: Email Prompt Builder

Below the campaign form, display a Prompt Section where the user defines how the AI should generate emails.

Field:

AI Email Generation Prompt
Text Area

Description:
The user writes instructions describing tone, style, and structure of the outreach email.

Example Prompt:

Write a short personalized cold email.

Tone: Friendly and professional  
Goal: Book a 15 minute demo  
Audience: SaaS founders  

Use personalization with first name and company name.  
Keep the email under 120 words.  
Include a clear call-to-action asking for a quick meeting.

--------------------------------------------------

Step 3: Generate Emails Button

Button Label:
Generate Personalized Emails

When clicked:

1. The system compiles:
   - Campaign Intent Form Data
   - Prompt
   - Selected Personalization Variables

2. A structured request is sent to Gemini API.

--------------------------------------------------
